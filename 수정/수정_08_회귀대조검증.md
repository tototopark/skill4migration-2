# 수정_08_회귀대조검증.md - Gate 2 회귀 테스트 통과 및 대조 검증 보고서

본 문서는 sitepro 레거시 마이그레이션 프로젝트의 **FastAPI 백엔드 이식**에 대한 최종 **Gate 2 회귀 대조 검증 및 테스트 수행 결과**를 정리한 문서입니다.

---

## 1. 검증 개요
* **검증 대상**: 
  * `src/domain.py` (비밀번호 호환성, IP 추출 로직)
  * `src/main.py` (FastAPI 엔드포인트 통합 제어)
  * `tests/test_integration.py` (통합 테스트 스위트)
* **테스트 툴**: pytest (Python 3.11.7)
* **검증 기준**: 
  * PHP bcrypt (`$2y$`) 해시와 Python bcrypt 간의 100% 호환성 확보
  * HTTP 프록시 및 클라이언트 헤더를 통한 올바른 IP 획득 여부
  * 비정상 접근 시 403 Forbidden 차단 검증
  * `/auth/login` 로그인 시나리오 및 `/dev/hints` 개발자 진단용 힌트 스위치 정상 작동

### 2.2 테스트 케이스별 통과 내역

1. **`test_php_bcrypt_compatibility` (통과)**
   * PHP bcrypt 해시 포맷(`$2y$`)을 동적으로 `$2b$` 포맷으로 정확히 교정 및 대응하는 검증기가 정상 작동함을 검증.
   * `dev12345` 퀵필 비밀번호 통과 검증 성공.

2. **`test_ip_extraction_logic` (통과)**
   * `x-forwarded-for`와 `http-client-ip` 등의 HTTP 헤더 순서별로 클라이언트 IP를 추출하는 로직의 정밀성 검증 성공.

3. **`test_home_page_access_denied` (통과)**
   * 허용되지 않은 IP와 기기 승인 키가 없을 시 접근이 엄격히 차단(403 Forbidden)되어 레거시 보안 통제 기능과 100% 일치함을 확인.

4. **`test_auth_login_api` (통과)**
   * 로그인 API의 로그인 실패 시나리오(401 Unauthorized 반환) 검증.

5. **`test_dev_hints_api` (통과)**
   * 개발자 진단용 힌트 스위치 활성화 시 메타데이터(API 라우트 정보 등) 바인딩 정상 작동 검증.
