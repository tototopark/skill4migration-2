# [sitepro 레거시 마이그레이션 이중 게이트 계획]

본 계획서는 레거시 PHP 프로젝트 `sitepro`를 모듈화된 백엔드(FastAPI 등) 및 신규 웹 스택으로 이전하기 위한 **code-migration-quality-gate** 스킬 기반의 Gate 1 마이그레이션 설계서입니다.

---

## User Review Required

> [!IMPORTANT]
> **마이그레이션 핵심 검토 사항**
> 1. **IP 접근 제어 및 디바이스 검증 보존**: `1.php`에 정의된 `tb_login` 테이블의 IP 검증 (`ip_1`, `ip_2`, `ip_3`)과 `tb_keys_remote_devices`의 `private_key` 검증 비즈니스 로직을 API 게이트웨이 또는 미들웨어 단에 100% 이식해야 합니다.
> 2. **사용자 권한 레벨 세분화 (`right_level`)**: 각 직원(Staff) 등급별 메뉴 접근 룰이 100% 동일하게 포팅되어야 합니다.
> 3. **데이터 무결성 및 SQLite 변환**: 레거시 CSV 데이터 구조(`SQL_DL_FILE-Dev` 폴더)를 활용해 로컬 SQLite 스키마를 구성하며 MySQL 호환성을 유지합니다.
